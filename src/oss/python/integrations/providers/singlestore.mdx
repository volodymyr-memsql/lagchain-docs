---
title: "SingleStore integrations"
description: "Integrate with SingleStore using LangChain Python."
---

[SingleStore](https://singlestore.com/) is a high-performance, distributed SQL database designed for both [cloud](https://www.singlestore.com/cloud/) and on-premises environments.

The `langchain-singlestore` package provides the following components:

- **`SingleStoreVectorStore`**: Store document embeddings and perform vector, full-text, and hybrid searches with multiple search strategies and scoring modes.
- **`SingleStoreLoader`**: Load documents directly from a SingleStore database table.
- **`SingleStoreSemanticCache`**: Use SingleStore as a semantic cache for LLM responses using vector similarity.
- **`SingleStoreChatMessageHistory`**: Store and retrieve chat message history in SingleStore.
- **`SingleStoreSQLDatabaseRetriever`**: Execute SQL queries against SingleStore and retrieve results as documents.

```python
from langchain_singlestore import (
    SingleStoreChatMessageHistory,
    SingleStoreLoader,
    SingleStoreSemanticCache,
    SingleStoreSQLDatabaseRetriever,
    SingleStoreVectorStore,
)
```
